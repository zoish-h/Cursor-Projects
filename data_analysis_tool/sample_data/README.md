# Sample datasets

Use these to try the Data Analysis Tool.

| File | Description |
|------|-------------|
| **sales_metrics.csv** | Sales by date, region, product with revenue, units, error_rate, and a binary target. Good for regression/time-series and classification-style metrics. |
| **customer_feedback.csv** | Survey responses: segment, satisfaction score, NPS category, date, comment length. Good for categorical and numeric distribution analysis. |

**How to use:** In the app (http://localhost:8000/app/), drag and drop one or more files from this folder, or click to browse and select them.
